# דוח דימונים — תוסף Chrome

## התקנה

1. פתח Chrome ועבור ל־`chrome://extensions/`
2. הפעל **מצב מפתחים** (Developer mode)
3. לחץ **טען פריט לא ארוז** (Load unpacked)
4. בחר את התיקייה `extention-giuer`

## שימוש

1. פתח בדף SharePoint את חלון הדיאלוג עם טבלת הדימונים (`role="dialog"`)
2. לחץ על אייקון התוסף בסרגל הכלים
3. לחץ **הורד דוח Word**

הקובץ `דוח דימונים.docx` יורד אוטומטית.

## קבצים

| קובץ | תפקיד |
|------|--------|
| `popup.html` / `popup.js` | כפתור ההורדה בחלון הקופץ |
| `content.js` | מזריק את סקריפט הייצוא לדף |
| `export-page.js` | לוגיקת הייצוא (זהה לסקריפט המקורי) |
| `index.umd.js` | ספריית docx (מקומית, ללא CDN) |
